# **==INSTALLATION==**

* Download content of the theme folder as as zip files.<br>
* Copy to your SD-Card in the ARCHIVE folder<br>
* Open Archive Manager on your device<br>
* Install archive<br>
* Set theme in the theme picker in settings<br>
